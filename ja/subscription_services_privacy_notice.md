# Mozilla サブスクリプションサービス | プライバシーに関する通知

発効日 2025 年 12 月 17 日

## Mozilla では、皆様のプライバシーを念頭に製品を設計しています。

__Mozilla VPN__は、お使いのデバイスのインターネット接続を保護します。Mozilla は Mullvad と提携し、インターネットトラフィックをプライベートに、かつ安全に暗号化します。

__Mozilla Monitor__は、お使いのアカウントのいずれかが既知のデータ侵害に遭遇していないかをモニターできます。一部の地域では、ウェブ中のさまざまな場所からユーザーの機密情報を検索し、削除するのを支援することもできます。 

__Firefox Relay__は、セキュアなメール機能と、一部の地域では電話番号のマスキングにより、身元情報を保護します。セキュアで使いやすいメールおよび電話番号のマスキング機能により、身元情報をプライベートに保つことができます。これにより、新しいアカウントに匿名でサインアップできるほか、迷惑メッセージや不要な電話を差し止め、受信トレイで必要なメールだけを受け取れるようにすることができます。  

プライバシーに関する本通知では、Mozilla VPN、Mozilla Monitor、Firefox Relay が収集・共有するデータは何か、およびそのデータを収集・共有する理由について説明します。また Mozilla では、情報をどのように受信、処理、および共有するかに関し、[Mozilla のプライバシーポリシー](https://www.mozilla.org/privacy/) に従っています。

## 留意事項

__Mozilla アカウントの情報。__これらのサービスを利用するには Mozilla アカウントが必要で、このアカウントがメールアドレス、ロケール、IP アドレスを Mozilla に送信します。[Mozilla アカウントデータの慣行](https://www.mozilla.org/privacy/mozilla-accounts/)の詳細をご覧ください。

__お支払い情報。__有料サブスクリプションが必要なサービスに登録した場合、皆様は、Mozilla のサードパーティー決済業者1社（Stripe、Apple、PayPal、またはGoogle Pay）を通じて支払いを行います。Mozilla は、皆様のアカウントの記録（請求先住所とお支払い方法の下4桁を含む）およびアカウントのサブスクリプションの状態を受信します。Mozilla が皆様のお支払い情報すべてを保管することはありません。

### Mozilla VPN {: #vpn }

__操作データ。__操作データの収集は、デフォルトでは無効になっています。オプトインした場合、Mozilla は Mozilla VPN とのやり取りに関するデータを受け取ります。たとえば、保護の対象から除外しているアプリがあるかどうか（どのアプリかに関する情報は含まれません）、Mozilla VPN のサブスクリプションを何台のデバイスで使用しているか、また広告ブロック機能を有効にしているかどうか、などの情報を把握できるようになります。テレメトリの詳細なレビューについては、「[VPN の操作データ](https://dictionary.telemetry.mozilla.org/apps/mozilla_vpn)」に関するページで確認できます。操作データは、ユーザーのためのパフォーマンスと安定性の改善、および サービスのパフォーマンス測定のために Mozilla が使用します。

さらに Mozilla では、マーケティングキャンペーンの効果を把握するため、キャンペーンデータや参照データ（どの広告キャンペーンに反応したか、使用しているオペレーティングシステム、デバイスの種類、デバイス識別子、OS のバージョンなど）に加えて、IP アドレスを受信する場合があります。Mozilla ではこの情報を Adjust と共有しますが、ユーザーの IP アドレスは共有も保存も行いません。[Adjust のドキュメント](https://github.com/mozilla-mobile/mozilla-vpn-client/blob/main/src/adjust/adjust.md) をお読みください。

__技術データ。__サービス全体において、IP アドレスはサーバーログの一環として一時的に収集され、90 日間保存されます。VPN については、Mozilla は、インストールされている Mozilla VPN クライアントソフトウェアのバージョン情報や、そのソフトウェアがインストールされているデバイスに関する情報（オペレーティングシステムやハードウェア構成を含む）も受け取ります。技術データは、ユーザーのためのパフォーマンスと安定性の改善、および サービスのパフォーマンス測定のために Mozilla が使用します。 

さらに、ユーザーがアプリケーション内でデータ収集に同意していた場合、エラーおよびパフォーマンスのモニタリングのために、Mozilla のパートナーである Sentry.io が技術的データを受け取ります。詳しくは、[Sentry.io のプライバシーポリシー](https://sentry.io/privacy/)をご覧ください。

__位置情報。__Mozilla は、サービスへのサインアップおよび使用時に IP アドレスを受信し、これをサーバーログの一環として 90 日間保存します。Mozilla サブスクリプションサービスの提供状況、価格、オファーが国によって異なる場合があるため、IP アドレスはユーザーのおおよその所在地を推定する目的で使用されます。また、不正や悪用を防ぐ目的でも、すべての製品でこれを使用しています。

__ネットワークデータ。__Mozilla VPN を有効にすると、インターネットトラフィックは暗号化され、Mullvad に送信されます。Mozilla VPN サービスを使用している間、いかなる場合においても、Mozilla およびパートナーの Mullvad のいずれも、ユーザーのネットワークアクティビティのログを保持することはありません。詳しくは、[Mullvad のプライバシーポリシー](https://mullvad.net/help/no-logging-data-policy/)をご覧ください。

### Mozilla Monitor {: #monitor }

__操作データ。__Mozilla は、サービスを使用する際のユーザーの操作に関するデータを受け取ります。たとえば、ログイン・ログアウトの状況や設定した各種環境設定などがこれに含まれます。詳しくは「[Monitor の操作データ](https://dictionary.telemetry.mozilla.org/apps/monitor_frontend)」をご覧ください。より一般的には、Mozilla は、Cookie、デバイス情報、および IP アドレスをクリア GIF、Cookie、第三者によるサービスとともに使用することもあります。これらは、ユーザーが Mozilla の製品、サービス、コミュニケーション、ウェブサイト、オンラインキャンペーン、スニペット、デバイス、およびその他のプラットフォームをどのように利用しているかを総体的に把握するのに役立ちます。Mozilla では、以下を使用しています。

* [HaveIbeenpwned。](https://haveibeenpwned.com/)HaveIbeenpwned は、個人データが侵害された可能性のある場所を精査するために、ハッシュ化バージョンのメールアドレスを受信します。HaveIbeenpwned は、新たなデータ侵害を継続的に精査するために、この個人データを保持します。ただし、HaveIbeenpwned が提供する[オプトアウトのメカニズム](https://haveibeenpwned.com/OptOut)を使用することで、メールアドレスの削除をリクエストすることができます。
    
* [Amazon Web Services (AWS)。](https://aws.amazon.com/privacy/)Mozilla Monitor では、Mozilla Monitor サービスに関連してユーザーにメールを送信するため、AWS を利用しています。これには、完全レポート、情報漏洩の警告、安全に関するヒントが含まれます。このデータは、Monitor のサブスクリプションを解除すると削除されます。

* [Formstack。](https://www.formstack.com/)Mozilla Monitor は、Formstack を使用して、ユーザーが任意で提供するフィードバックを収集しています。Formstack のプライバシー慣行について詳しくは、[Formstack プライバシーポリシー](https://www.formstack.com/legal)をご覧ください。
  
* [Google アナリティクス。](https://marketingplatform.google.com/about/analytics/)Mozilla Monitor は、ユーザーが Mozilla のウェブサイトをどのように利用しているかを把握するためのメトリクスを得るために、Google アナリティクスを使用しています。この機能は、Mozilla がウェブサイトのコンテンツを改善するのに役立ちます。Google による個人情報の利用について詳しくは、[Google アナリティクスの「プライバシーとセキュリティ](https://support.google.com/analytics/topic/2919631?&ref_topic=1008008&sjid=14989286036636170427-NA)」をご覧ください。[Google アナリティクス オプトアウト ブラウザ アドオン](https://tools.google.com/dlpage/gaoptout)をインストールすることで、本サービスへのアクセスに関するデータの収集を防止し、Google アナリティクスへのデータ送信を禁止することができます。
 
### Firefox Relay {: #relay }

__操作データ。__Mozilla は、Firefox Relay の操作状況に関するデータ（たとえば、ログイン・ログアウトの状況や設定した各種環境設定など）を受け取ります。[Relay 操作データ](https://github.com/mozilla/fx-private-relay/blob/main/METRICS.md)の詳細については、こちらをご覧ください。なお、ブラウザで「[Do Not Track（DNT）](https://support.mozilla.org/kb/how-do-i-turn-do-not-track-feature)」機能を有効にすることで、このデータ収集をオプトアウトすることができます。 

より一般的には、Mozilla は、Cookie、デバイス情報、および IP アドレスを Cookie や第三者によるサービスとともに使用することもあります。これらは、ユーザーが Mozilla の製品、サービス、コミュニケーション、ウェブサイト、オンラインキャンペーン、スニペット、デバイス、およびその他のプラットフォームをどのように利用しているかを総体的に把握するのに役立ちます。Mozilla では、以下を使用しています。

* Cookie をユーザのデバイスに配置し、ユーザーが Mozilla のウェブサイトにどのように関与しているかのメトリクスを得るための Google アナリティクス。この機能は、Mozilla がウェブサイトのコンテンツを改善するのに役立ちます。Google アナリティクスが Mozilla Monitor の使用に関するデータを収集しないようにするには、[Google アナリティクス オプトアウト ブラウザ アドオン](https://tools.google.com/dlpage/gaoptout)をインストールすることができます。

__技術データ。__Mozilla は、インストールされている Relay アドオンソフトウェアのバージョン情報や、そのソフトウェアがインストールされているデバイスに関する基本的な情報（オペレーティングシステムやハードウェア構成を含む）を受け取ります。サービス全体において、IP アドレスはサーバーログの一環として一時的に収集され、90 日間保存されます。技術データは、ユーザーのためのパフォーマンスと安定性の改善、および サービスのパフォーマンス測定のために Mozilla が使用します。

__位置情報。__Mozilla は、本サービスの登録および使用時に、ユーザーの IP アドレスを受信します。Mozilla サブスクリプションサービスの提供状況、価格、オファーが国によって異なる場合があるため、IP アドレスはユーザーのおおよその所在地を推定する目的で使用されます。また、不正や悪用を防ぐ目的でも、すべての製品で IP アドレスを使用しています。

__Firefox Relay のメールメッセージ。__マスキングされたメールアドレスからユーザーが主に使用しているメールアドレスへメールメッセージを送信および転送するため、Firefox Relay はユーザーのメールメッセージを処理します。メッセージの内容をこちらが読み取ったり、保存したりすることはありません。メールをユーザー宛に配信できなかった場合、そのメールはサーバー上に一時的に保管され、配信後に削除されます（3 日を超えて保持されることはありません）。この機能を使用してプロモーションメールをブロックすると、本サービスはメールヘッダーを確認して、そのメールをブロックすべきかどうかを判断するします。

__Firefox Relay のマスク（およびその使用場所）。__Mozilla では、サービスを提供するためにユーザーのアカウント情報のコピーを保持しており、特に、主要なメールアドレスとマスキングされたメールアドレスを関連付ける目的でこれを使用します。ユーザーがカスタムマスクを作成した場合、Mozilla はそのマスク宛にメールを転送するために、そのマスクを保存します。Mozilla は、ユーザーがマスクを作成したサイト、それ以降にマスクを使用したサイト、ならびにマスクに関連付けられたラベルを、ユーザーが使用したいときにマスクを簡単に見つけられるようにする目的で保存します。[この機能を有効・無効にする方法](https://relay.firefox.com/faq)などの詳細はこちらをご覧ください。

__Firefox Relay の電話通話とテキストメッセージ。__電話通話やテキストメッセージを送信および転送するため、Firefox Relay はユーザーの電話番号とテキストメッセージを処理します。通話やテキストメッセージの履歴を表示し、テキスト返信を送信し、電話番号をブロックできるようにするため、Relay を通じてユーザーが連絡した電話番号のログを保存しています。ただし、Firefox Relay を通じてかけた電話通話の内容をモニターしたり保存したりすることはありません。[この機能を有効・無効にする方法](https://relay.firefox.com/faq)などの詳細はこちらをご覧ください。

__共有する情報。__Firefox Relay では、サービスをユーザーに提供するために、第三者と情報を共有します。Mozilla では、ユーザーの情報を保護することを義務付ける契約をこれらの会社と締結しています。Firefox Relay をサポートするために利用している会社は次のとおりです。

* __[Amazon Web Services 。](https://aws.amazon.com/privacy/)__Amazon Web Services（AWS）は、クラウドコンピューティングプラットフォームです。Firefox Relay では、マスキングされたメールアドレス宛に送信されたメールを受信し、それらを Mozilla アカウントに関連付けられた主なメールアドレスへ転送するために、AWS を利用しています。ユーザーの主なメールアドレスとマスキングされたメールアドレスとの関連付けを把握しているのは、Mozilla だけです。

* __[Twilio。](https://www.twilio.com/en-us/legal/privacy)__Twilio は、ユーザーの電話番号、電話番号のマスク、および通話やテキストメッセージをやり取りする相手の電話番号を受け取ります。Twilio は、Firefox Relay を通じてユーザーが送受信するテキストメッセージの内容も受信します。また、Mozilla は、Firefox Relay を通じて送受信されるテキストメッセージについて、Twilio サービスがその記録を 7 日後に削除するよう設定しています。
