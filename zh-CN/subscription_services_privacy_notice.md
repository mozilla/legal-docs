# Mozilla 订阅服务 | 隐私声明

2025 年 12 月 17 日生效
{: datetime="2025-12-17" }

## Mozilla 在设计产品时始终顾及和考虑用户的隐私权。

__Mozilla VPN__保护您设备的互联网连接安全。Mozilla 与 Mullvad 合作，为您的互联网流量提供私密且安全的加密保护。

__Mozilla Monitor__可监测您的任意帐户是否涉及已知数据泄露事件。在部分地区，它还能帮助您扫描并清除网络各处留存的个人信息。 

__Firefox Relay__通过安全电子邮箱及部分地区的电话号码掩蔽功能保护您的身份。我们安全便捷的电子邮箱与电话号码隐藏服务可保障身份隐私，助您匿名注册新帐户、拦截垃圾短信与骚扰电话，仅接收所需邮件。  

本隐私声明说明了 Mozilla VPN、Mozilla Monitor 以及 Firefox Relay 会收集和共享哪些数据，以及这么做的原因。此外，我们遵守 [Mozilla 隐私政策](https://www.mozilla.org/privacy/)，以作为我们如何接收、处理和共享信息的准则。

## 须知

__Mozilla 帐户信息。__这些服务需要一个 Mozilla 帐户，该帐户会向 Mozilla 发送您的电子邮箱地址、区域设置以及 IP 地址。了解有关 [Mozilla 帐户数据实践](https://www.mozilla.org/privacy/mozilla-accounts/)的更多信息。

__付款信息。__当您订阅需要付费的服务时，您将通过我们的第三方支付提供商之一进行付款：Stripe、Apple、PayPal 或 Google Pay。Mozilla 会收到您的帐户记录（包括您的账单地址和付款方式的最后四位数字）以及您的帐户订阅状态。Mozilla 不会存储您的全部付款详情。

### Mozilla VPN {: #vpn }

__交互数据。__交互数据收集默认处于禁用状态。如果您主动选择开启该功能，Mozilla 将接收您使用 Mozilla VPN 时产生的交互行为数据。例如，我们会获取这类信息：您是否将某些应用程序排除在保护范围之外（但不会知晓具体是哪些应用程序）、您的 Mozilla VPN 订阅服务同时在多少台设备上使用，以及您是否开启了广告拦截功能。如需全面了解我们的遥测数据，请参阅 [VPN 交互数据](https://dictionary.telemetry.mozilla.org/apps/mozilla_vpn)。我们使用交互数据来为用户提升性能与稳定性，并衡量服务的表现。

此外，为助力评估营销活动的效果，Mozilla 除了会获取营销活动数据和引荐数据（例如您参与过的广告活动、使用的操作系统、设备类型、设备标识符以及操作系统版本）外，还可能获取您的 IP 地址。Mozilla 会将这些数据共享给 Adjust，但不会共享或存储您的 IP 地址。请阅读 [Adjust 文档](https://github.com/mozilla-mobile/mozilla-vpn-client/blob/main/src/adjust/adjust.md)。

__技术数据。__在本服务中，您的 IP 地址会作为服务器日志的一部分被临时收集，存储期限为 90 天。针对 VPN 服务，Mozilla 还会获取您所安装的 Mozilla VPN 客户端软件版本信息，以及安装这些软件的设备信息，包括设备的操作系统和硬件配置。我们使用技术数据来为用户提升性能与稳定性，并衡量服务的表现。 

此外，如果您已在应用程序内同意数据收集，我们的合作伙伴 Sentry.io 将接收技术数据，用于错误排查与性能监控。详情请参阅 [Sentry.io 的隐私政策](https://sentry.io/privacy/)。

__位置数据。__当您注册并使用服务时，Mozilla 会获取您的 IP 地址，该地址会作为服务器日志的一部分保存 90 天；我们利用 IP 地址估算您的所在位置，因为 Mozilla 订阅服务的可用性、定价与优惠活动可能会因您所在的国家/地区而有所不同。我们还利用该信息在所有产品中防范欺诈和滥用行为。

__网络数据。__当您激活 Mozilla VPN 时，它会加密您的互联网流量并将其发送至 Mullvad。无论在何种情况下使用 Mozilla VPN 服务，Mozilla 及其合作伙伴 Mullvad 均不会留存您的任何网络活动日志。详情请参阅 [Mullvad 的隐私政策](https://mullvad.net/help/no-logging-data-policy/)。

### Mozilla Monitor {: #monitor }

__交互数据。__Mozilla 会收集您与本服务的交互行为数据。例如您的登录与登出行为、您设置的首选项等；详情请参阅 [Monitor 交互数据](https://dictionary.telemetry.mozilla.org/apps/monitor_frontend)。除此之外，Mozilla 还可能使用 Cookie、设备信息和 IP 地址，以及网络信标、Cookie 和第三方服务，帮助我们从整体上了解用户如何与我们的产品、服务、通信、网站、在线广告活动、片段、设备以及其他平台进行交互。我们使用：

* [HaveIbeenpwned。](https://haveibeenpwned.com/)HaveIbeenpwned 会获取您电子邮箱地址的哈希处理版本，用于扫描检测您的个人数据是否可能已遭泄露。HaveIbeenpwned 会留存该个人数据，以便持续扫描是否出现新的数据泄露事件；您可以通过 HaveIbeenpwned 的[退出机制](https://haveibeenpwned.com/OptOut)，申请将自己的电子邮箱地址从该平台中移除。
    
* [Amazon Web Services (AWS)。](https://aws.amazon.com/privacy/)Mozilla Monitor 采用 AWS 以便向您发送与 Mozilla Monitor Service 相关的电子邮件，内容包括完整报告、泄露提醒以及安全提示。当您取消 Monitor 订阅时，此数据将被删除。

* [Formstack。](https://www.formstack.com/)Mozilla Monitor 借助 Formstack 来收集用户自愿提供的反馈信息。有关 Formstack 隐私实践的信息，请参阅 [Formstack 隐私政策](https://www.formstack.com/legal)。
  
* [Google Analytics。](https://marketingplatform.google.com/about/analytics/)Mozilla Monitor 使用 Google Analytics 来获取有关用户如何与我们网站互动的指标数据。这可以帮助我们改善网站内容。如需了解 Google 如何使用您的个人信息，请访问 [Google Analytics 的隐私和安全](https://support.google.com/analytics/topic/2919631?&ref_topic=1008008&sjid=14989286036636170427-NA)。您可以安装 [Google Analytics Opt-out Browser Add-on](https://tools.google.com/dlpage/gaoptout)，以阻止收集有关您访问本服务的数据，并禁止向 Google Analytics 传输数据。
 
### Firefox Relay {: #relay }

__交互数据。__Mozilla 会收集您与 Firefox Relay 的交互行为数据，例如您的登录与登出操作、您设置的首选项等。进一步了解 [Relay 交互数据](https://github.com/mozilla/fx-private-relay/blob/main/METRICS.md)；您可以通过在浏览器中开启[“请勿跟踪”(DNT)](https://support.mozilla.org/kb/how-do-i-turn-do-not-track-feature) 功能来选择退出该数据收集机制。 

除此之外，Mozilla 还可能使用 Cookie、设备信息和 IP 地址，以及 Cookie 和第三方服务，帮助我们从整体上了解用户如何与我们的产品、服务、通信、网站、在线广告活动、片段、设备以及其他平台进行交互。我们使用：

* Google Analytics，在您的设备上放置 Cookie，以获取关于用户如何与我们的网站交互的指标。这可以帮助我们改善网站内容。若要阻止 Google Analytics 收集您使用 Mozilla Monitor 时的相关数据，您可以安装 [Google Analytics Opt-out Browser Add-on](https://tools.google.com/dlpage/gaoptout)。

__技术数据。__Mozilla 会收集您所安装的 Relay 附加组件版本，以及安装附加组件的设备信息，包括设备的操作系统和硬件配置。在本服务中，您的 IP 地址会作为服务器日志的一部分被临时收集，存储期限为 90 天。我们使用技术数据来为用户提升性能与稳定性，并衡量服务的表现。

__位置数据。__当您注册并使用服务时，Mozilla 会获取您的 IP 地址。我们会使用 IP 地址来大致判断您的地理位置，因为 Mozilla 订阅服务的可用性、定价与优惠活动可能会因您所在的国家/地区而有所不同。我们还使用您的 IP 地址在所有产品中防范欺诈和滥用行为。

__Firefox Relay 电子邮件内容。__为了将您的电子邮件内容从您的别名电子邮箱地址转发至主邮箱地址，Firefox Relay 会对您的电子邮件内容进行处理。我们不会读取或存储您任何邮件的内容。如果电子邮件无法投递给您，我们会将其暂存于服务器，并在成功投递后将其删除（无论如何，我们保留邮件的时间都不会超过三天）。如果您启用拦截营销邮件的服务，该服务会检查电子邮件标头信息，以此判定邮件是否应被拦截。

__Firefox Relay 别名（及其使用位置）。__为提供服务，Mozilla 会保留您帐户信息的副本，特别是将您的主邮箱地址与您的别名邮箱地址相关联。如果您创建了自定义别名，Mozilla 会存储该别名，以便将电子邮件转发至该别名。为了确保您在需要使用时能轻松找到您的别名，Mozilla 会存储您创建别名的网站、您后续使用别名的网站，以及与别名关联的任何标签。了解如何[启用和禁用这些功能](https://relay.firefox.com/faq)。

__Firefox Relay 电话和短信。__为了发送和转发您的电话和短信，Firefox Relay 会处理您的电话号码和短信内容。我们会存储您通过 Relay 联系过的电话号码日志，以便显示您的通话和短信日志、发送短信回复以及屏蔽电话号码。我们不会监控或存储您通过 Firefox Relay 拨打的通话内容。了解如何[启用和禁用这些功能](https://relay.firefox.com/faq)。

__我们共享的信息。__Firefox Relay 会与第三方共享信息，以便为您提供服务。Mozilla 已与这些公司签订协议，要求其保护您的信息安全。以下是为 Firefox Relay 提供支持的合作方名单：

* __[Amazon Web Services。](https://aws.amazon.com/privacy/)__Amazon Web Services (AWS) 是一个云计算平台。Firefox Relay 使用 AWS 来接收发送至您别名邮箱地址的邮件，并将其转发至与您的 Mozilla 帐户关联的主邮箱地址。只有 Mozilla 知晓您的主邮箱地址与别名邮箱地址之间的关联。

* __[Twilio。](https://www.twilio.com/en-us/legal/privacy)__Twilio 会接收您的电话号码、您的电话别名，以及您与之通话和收发短信的电话号码。Twilio 还会接收您通过 Firefox Relay 发送和接收的短信内容；Mozilla 已将 Twilio 服务设置为在 7 天后删除您通过 Firefox Relay 发送和接收的短信记录。
